# Steve's Advanced Storage

A [NeoForge](https://neoforged.net/) mod for Minecraft 1.21.1 that adds advanced storage features on top of [Steve's Simple Storage](https://github.com/scuba10steve/steves-storage-system).

## Features

- **Advanced Storage Core** — multiblock controller that ties together all advanced storage components
- **Block Storage I** — high-capacity storage rack that integrates into the multiblock
- **Recipe Memory Box** — stores and recalls crafting recipes for the storage network
- **Advanced Statistics** — detailed breakdown of storage capacity, power draw, and block counts by tier

## Requirements

- Minecraft 1.21.1
- NeoForge 21.1.x
- [Steve's Simple Storage](https://github.com/scuba10steve/steves-storage-system)
